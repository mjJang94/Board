# 메모장

MVVM 패턴을 기반으로 사용하고 싶은 라이브러리를 조합하여 만든 간단 메모앱

## 라이브러리
- DataBinding   
- Coroutine   
- Koin   
- Room   
- LiveData   
- admob

## 개발환경
- Android Studio   
- Kotlin   
- Git
